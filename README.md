## Run Locally

Clone the project

```bash
  git clone git@github.com:yudhaananda/dating-app.git
```

Go to the project directory

```bash
  cd dating-app
```

Create `.env` file like this:

```shell
DB_USER=
DB_PASS=
DB_PORT=
DB_HOST=
DB_NAME=
JWT_SECRET_TOKEN=
DB_TYPE= mysql
```

Install initialize go work

```bash
  go work init
  go work use $PWD
```

Auto migrate DB

```bash
  cd docs/sql
  sh init.sh
```

Start the server

```bash
  make run
```
